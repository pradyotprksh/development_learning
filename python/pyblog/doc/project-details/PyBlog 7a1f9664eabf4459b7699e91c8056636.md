# PyBlog

A blog application made using python and firebase as a backend service.

---

## Things to learn:

- [x]  Python
- [ ]  GUI
- [ ]  Firebase in python
    - [ ]  Login
    - [ ]  Firestore
    - [ ]  Storage

## **Features**:

- Login
- Profile
- Create Blog
    - Tags to Blog
    - Tag other users
    - Markdown Feature
- See All Blogs
- Follow Others
- Subscribe To A User
    - Get Emails for New Blog
- Search
    - User
    - Blog
